Algunos productos

En RST nos hemos enfocado principalmente en lo que son los cojinetes, y a partir de allí, nos hemos expandido en los diversos tipos de productos.

Bolas

Diseñado pensando en una carga ligera y revoluciones altas, consta con una o mas hileras de bolas.

Rodillos

diseñados para soportar carga gracias a su diseño, este puede venir con rodillos cónicos o cilíndricos.

Chumaceras

Soporte de eje, consta con una base que puede ser de banco, pared, tensora, etc.

Rotulas

con o sin vástago, hembras o machos, diseñadas para la alineación de ejes

Agujas

rodillos delgados unidos a una jaula, puede contener bujes internos, externos o ambos, diseñados para poco espacio, carga y velocidad media

Automotriz

Bufas con o sin ABS, muñones, tripoides, cruces, rodamientos piloto, para manubrio, caja, entre otros

Transmision

Acoples, sprocket o polea de una o mas hileras, para cadena o faja, ideal para la transmisión de potencia

Retenedores

De metal, nitrilo, o viton, ideales para retener grasa o aceite.
